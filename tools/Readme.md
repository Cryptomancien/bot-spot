# Telegram

1 - Edit bot.conf
```bash
TELEGRAM=1
TELEGRAM_BOT_TOKEN=""
TELEGRAM_CHAT_ID=""
```

2 - Search @BotFather

3 - /newbot

4 - Get token like that XXX:AAA_BBB

5 - Edit bot.conf

6 - Open browser and go to https://api.telegram.org/bot{our_bot_token}/getUpdates (replace with your token)

7 - Get chat_id and edit bot.conf
